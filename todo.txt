Pocket Wallet Plan (macOS focus)
================================

1) Key & address basics
   [x] Generate ed25519 keys from mnemonic/seed
   [x] Bech32 encode addresses (pc/tpc)
   [x] Encrypted keystore (password + Argon2id + AES-GCM)
   [x] Export/import (seed phrase)

2) Chain RPC integration (via Lantern)
   [x] Query balance: /getBalance
   [x] Chain status: /weave/chain/head, /getDifficulty
   [x] Bearer auth handling (server-side via headers/env)

3) Transaction building & signing
   [x] Build/sign/submit transfer
   [x] Build/sign/submit stake
   [x] Build/sign/submit unbond
   [x] Build/sign/submit update-validator
   [x] Nonce helper uses pending pool (not just ledger nonce)

4) Mining payout support for Weave
   [x] Select/store payout address and attestation token
   [ ] Simple API/IPC for Weave to fetch payout address (beyond current config/profile files)

5) UI/UX surface (macOS)
   [x] Tauri GUI scaffold (init/show/balance/head/difficulty)
   [x] Send form for transfers
   [x] Tx kind selector + dynamic fields (stake/unbond/update-validator)
   [x] Lock/unlock flow (clear in-memory password, disable actions, auto-lock timer)
   [x] Persist GUI settings (RPC/token/payout) and prefill on load
   [x] Error banners and validation on inputs

6) Security & storage
   [ ] Hardware keystore hook (future)
   [ ] Document PBKDF/crypto params (Argon2id + AES-GCM)

7) Config & env
   [x] Rate-limit/backoff for RPC (+ TLS strict by default; POCKET_TLS_INSECURE to override)
   [x] Ensure HRP/chain-id alignment checks on sends

8) Tests & tooling
   [x] Unit tests: key derivation/address encoding + build_kind serialization
   [x] Integration tests against local Lantern RPC (mock or real)
   [x] Lint/format CI
   [x] Deterministic signing property test (signature-agnostic bytes)

9) Packaging & deployment (macOS)
   [x] Tauri bundle for macOS (.app/.dmg) via CI artifact (unsigned)
   [x] App icon placeholder added (gui/src-tauri/icons/icon.png); signing/notarization plan doc
   [ ] Release pipeline (publish artifacts + notarized build when creds provided)
